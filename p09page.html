Types of Cloud Computing (Deployment Models):-

1. Public Cloud :

• Services are provided over the internet

• Available to anyone (public use)

• Cost-effective and scalable

Example: Amazon Web Services

2. Private Cloud :

• Used by a single organization only

• Offers better security and control

• Mostly used by large companies or government organizations

3. Hybrid Cloud :

• Combination of public and private cloud

• Allows data and applications to move between both

• Provides flexibility and better performance
